## 04 부분집합

부분집합 powerset

```
{1,2,3}의 부분집합: 8개
3C0		{},
3C1		{1}, {2}, {3},
3C2 	{1,2}, {2,3}, {1,3},
3C3		{1,2,3}
```

부분집합은 각 원소를 선택/제외하는 두 가지 경우를 가지고 조합하기 때문에 이는 결국 이진 트리로 구성할 수 있다. 비트 연산으로도 가능.



```python
N = 3
for i in range(2**N):
    for j in range(N):
        if i & (1 << j):
            func()
```
